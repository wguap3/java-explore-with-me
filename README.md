# ExploreWithMe

> **Проект выполнен как дипломная работа в рамках курса Java-разработки в Яндекс Практикуме.**

---

## О чём проект

Многомодульное backend-приложение, состоящее из двух микросервисов:
- **Сервис событий и пользователей** — основная логика
- **Сервис аналитики** — сбор статистики по просмотрам и действиям

Сервисы взаимодействуют через REST API, каждый работает в своём Docker-контейнере с отдельной PostgreSQL-базой.

---
## Важно
Весь backend-код (сервисы, контроллеры, репозитории, DTO, мапперы, конфигурации, Docker-сборка) написан мной **с нуля**.

Для автоматической проверки заданий использовались **стандартные тесты платформы Яндекс Практикума** — они **не являются частью моей реализации** и служат только для валидации соответствия требованиям.

---
